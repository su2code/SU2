Constructor of the class.

Parameters
[in]valMat_Model- Type of material model (i.e. numerics) for the element, see FEA_TERM etc. in option_structure.hpp.
[in]valMat_Prop- Index of the physical properties (E,nu,rho,rho_dead_load) assigned to the element.
[in]valElectric_Prop- Index of the electric properties.
[in]valDV- Index of the design variable assigned to the element (bound to a material property by "DESIGN_VARIABLE_FEA").
[in]valDensity- Value for Design and Physical densities (topology optimization variables).
