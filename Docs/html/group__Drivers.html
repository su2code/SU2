Detailed Description

Group of classes which define the iterative process used to converge the equations (fluid, turbulence, elasticity, FSI, CHT, etc.). In general, "Driver" classes use "Iteration" classes to advance one inner iteration, in turn "Iteration" classes use "Integration" classes to perform space and time integration. The latter use mostly the "Solvers".
