class  CRoeBase< Derived, Base >
 Base class for Roe schemes, derived classes implement the dissipation term in a const "finalizeFlux" method. A base class implementing "viscousTerms" is accepted as template parameter. Similarly to derived, that method should update the flux and Jacobians, but whereas "finalizeFlux" is passed data prepared by CRoeBase, "viscousTerms" takes the same input arguments as "ComputeFlux", i.e. it can fetch more data from CVariable. Derived is meant to implement small details, Base is meant to do heavy lifting. More...
