Detailed Description

template<class T>
class C3DDummyMiddleView< T >

Helper class, adds dummy middle dimension to a reference of a matrix object making it a dummy 3D array.

Note
The constness of the object is derived from the template type, but we allways keep a reference, never a copy of the associated matrix.
