Detailed Description

template<class Vector_t = const su2double*, class Matrix_t = const Vector_t*>
struct CNumerics::ResidualType< Vector_t, Matrix_t >

Return type used in some "ComputeResidual" overloads to give a const-view of the internally stored flux vector and Jacobians to the outside.

Note
The default template params make this an "all const" object, it cannot change after instantiation, nor can it be used to change the data.
