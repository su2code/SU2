53 /*!
54  * \class CSysSolve
55  * \ingroup SpLinSys
56  * \brief Class for solving linear systems using classical and Krylov-subspace iterative methods
57  *
58  * The individual solvers could be stand-alone subroutines, but by
59  * creating CSysSolve objects we can more easily assign different
60  * matrix-vector products and preconditioners to different problems
61  * that may arise in a hierarchical solver (i.e. multigrid).
62  *
63  * The methods of this class are designed to be called by multiple OpenMP threads.
64  * Beware of writes to class member variables, for example "Residual" should only
65  * be modified by one thread.
66  */
