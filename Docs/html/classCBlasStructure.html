Function, to carry out the axpy operation, i.e y += a*x.

Parameters
[in]n- Number of elements in the vectors x and y.
[in]a- Specifies the scalar a.
[in]x- Array, which must be added, size of x must be at least 1 + (n-1)*abs(incx). param[in] incx - Specifies the increment of x. param[inout] y - Vector to be updated, size of y must be at least 1 + (n-1)*abs(incy). param[in] incy - Specifies the increment of y.
