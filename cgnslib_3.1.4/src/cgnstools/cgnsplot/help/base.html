Base Selection

The base selection entry allows the desired CGNS base to be selected for display. Clicking Mouse Button-1 in the entry field or on the pull-down arrow, will invoke a selection list displaying all the bases in the CGNS file. When a base is selected, the graphics will be reset, and the Display List will be filled with the list of zones under the base. If there is only 1 base in the CGNS file, then this will be disabled.
